# find-me-a-coffee-app
Find Me a Coffee é uma aplicação Java desenvolvida para ajudar usuários a encontrar cafeterias que atendam às suas necessidades específicas — como espaços que oferecem coworking, opções de alimentação com restrições (como vegana ou sem glúten), áreas pet friendly, entre outros diferenciais.

## ☕ Find Me a Coffee

**Find Me a Coffee** é uma aplicação Java criada para facilitar a busca por cafeterias que ofereçam mais do que apenas um bom café. Com foco em atender diferentes estilos de vida, o app permite filtrar os estabelecimentos com base em características como:

- Espaço para coworking
- Opções de alimentação com restrição (vegana, sem glúten, etc.)
- Ambiente pet friendly
- Conexão Wi-Fi, tomadas disponíveis e muito mais

## 🧠 Tecnologias e Conhecimentos Utilizados

- **Java** — linguagem principal do projeto
- **NetBeans IDE** — ambiente de desenvolvimento
- **Programação Orientada a Objetos (POO)**
- **Interface Gráfica com Swing** *(ou JavaFX, se for o caso)*
- **Consumo de API** — integração com APIs externas para buscar dados (ex: localização, estabelecimentos)
- **Manipulação de dados JSON**
- **Tratamento de exceções**
- **Estruturação de projeto e organização de código**
- **Boas práticas de versionamento com Git e GitHub**
